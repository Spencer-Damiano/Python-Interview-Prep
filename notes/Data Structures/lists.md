# Lists/Arrays Cheat Sheet

## Basics

# Creation & Access

```python
# creating
lst = []
lst = list()
lst = [1, 2, 3]
lst = list(range(5))

# Access
lst[0]   # First Item
lst[-1]  # Last Item
lst[-2]  # Second to last
lst[0:3] # Items 0-2
lst[:3]  # First 3 Items
lst[3:]  # 3 until the end of the list
lst[:]   # Copy of the entire list
```
